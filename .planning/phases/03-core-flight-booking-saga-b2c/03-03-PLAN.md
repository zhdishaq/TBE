<objective>
Deliver FLTB-06 (TTL extraction + monitor), FLTB-07 (compensation completeness via schedule-backed hard-timeout), COMP-05 (secrets in env/vault only), and COMP-06 (OpenTelemetry PII/PCI span scrubbing) — all cross-cutting concerns layered on top of the saga (03-01) and payment (03-02) skeletons.

Concretely:
1. Shared primitives in `TBE.Common`: `IEncryptionKeyProvider` + `EnvEncryptionKeyProvider` + `AesGcmFieldEncryptor` (D-08); `SensitiveAttributeProcessor` + `AddTbeOpenTelemetry` DI extension (D-09).
2. Fare-rule parsing pipeline in `BookingService.Application.Ttl`: `IFareRuleParser` with primary regex + per-GDS keyed adapters (Amadeus/Sabre/Galileo) and 2-hour fallback per D-07.
3. `CreatePnrConsumer` that consumes `CreatePnrCommand`, calls the Phase 2 FlightConnectorService, extracts TTL via `IFareRuleParser`, and publishes `PnrCreated(BookingId, Pnr, TicketingDeadlineUtc, ...)`.
4. `TtlMonitorHostedService` (5-minute poll) that emits `TicketingDeadlineApproaching` advisory events for 24h and 2h thresholds — consumed by 03-04 for notification dispatch.
5. Wiring: register `AddTbeOpenTelemetry` in every service's Program.cs (BookingService, PaymentService, NotificationService, FlightConnectorService, PricingService); wire `[Authorize]` FallbackPolicy where not already set; move every secret out of appsettings.*.json into env vars documented in `.env.example`.

**NOTE on FLTB-07 (Compensation):** The compensation chain code is already implemented in 03-01. This plan adds the infrastructure pieces FLTB-07 also requires — specifically the TTL-fired hard-timeout compensation path (D-04) which depends on the fare-rule parser producing a correct `TicketingDeadlineUtc` at PNR creation. Without this plan, the saga's `HardTimeout` schedule fires at an arbitrary default delay instead of at the real deadline.

Purpose: Ship compliance and TTL hardening as a dedicated Wave-3 plan after saga + payment exist, because it touches every service's Program.cs (sequential contention) and needs the saga to be published to real-world schedules.

Output: 7 new files in TBE.Common, 5 new files in BookingService.Application.Ttl, `CreatePnrConsumer.cs`, Program.cs updates on 5 services, `.env.example` created at solution root, and ~4 unit test files plus fare-rule fixture captures.
</objective>

    public sealed class CreatePnrConsumer(
        IHttpClientFactory http,
        IFareRuleParser parser,
        ILogger<CreatePnrConsumer> log) : IConsumer<CreatePnrCommand>
    {
        public async Task Consume(ConsumeContext<CreatePnrCommand> ctx)
        {
            var cmd = ctx.Message;
            try {
                // 1. Call FlightConnectorService (Phase 2 named client)
                var client = http.CreateClient("flight-connector");
                var response = await client.PostAsJsonAsync("/pnr", new { cmd.BookingId, cmd.OfferToken, cmd.PassengerRefs }, ctx.CancellationToken);
                response.EnsureSuccessStatusCode();
                var result = await response.Content.ReadFromJsonAsync<CreatePnrResult>(cancellationToken: ctx.CancellationToken)
                    ?? throw new InvalidOperationException("FlightConnectorService returned no body");
                // 2. Parse ticketing deadline
                DateTime deadlineUtc;
                if (!parser.TryParse(result.GdsCode, result.RawFareRule, out deadlineUtc)) {
                    // D-07 fallback
                    deadlineUtc = DateTime.UtcNow.AddHours(2);
                    await ctx.Publish(new FareRuleParseFailedAlert(cmd.BookingId, result.GdsCode, Sha256Hex(result.RawFareRule), DateTimeOffset.UtcNow));
                }
                await ctx.Publish(new PnrCreated(cmd.BookingId, result.Pnr, deadlineUtc, DateTimeOffset.UtcNow));
            } catch (HttpRequestException ex) {
                await ctx.Publish(new PnrCreationFailed(cmd.BookingId, $"gds http error: {ex.Message}"));
            } catch (Exception ex) {
                await ctx.Publish(new PnrCreationFailed(cmd.BookingId, ex.Message));
                throw;  // let MassTransit retry
            }
        }
        private static string Sha256Hex(string s) => Convert.ToHexString(SHA256.HashData(Encoding.UTF8.GetBytes(s ?? string.Empty)));
        private record CreatePnrResult(string GdsCode, string Pnr, string RawFareRule);
    }
