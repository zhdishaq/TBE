<objective>
Build the MassTransit orchestration saga that is the spine of Phase 3, on top of a newly created test infrastructure (Wave 0). This plan delivers: (0) the test projects + shared fixtures that every subsequent plan's automated commands depend on, (a) the canonical saga event + command contracts in TBE.Contracts (including SagaDeadLetterRequested), (b) the BookingSagaState entity + state machine in BookingService.Application (including Warn24HSent / Warn2HSent flags owned here — consumed by 03-03 TTL monitor, mutated here only), (c) EF Core persistence with optimistic concurrency per D-01, (d) the forward step chain BookingInitiated → PriceReconfirmed → PNRCreated → PaymentAuthorized → TicketIssued → PaymentCaptured → BookingConfirmed per D-05, (e) the full compensation chain with reverse-order rollback per D-03/D-05, (f) the public BookingsController for POST/GET and customer-facing read endpoints per D-21, (g) blanket [Authorize] enforcement per COMP-04, and (h) the SagaDeadLetter table for unrecoverable failures.

This plan intentionally does NOT implement Stripe (03-02), the TTL monitor/parser (03-03), or notifications (03-04). It emits commands onto RabbitMQ that later plans' consumers will handle; stub compensation consumers are limited to VoidPnr (called internally by the saga's compensation chain — the actual GDS void call is delegated via a command to FlightConnectorService from Phase 2).

Purpose: Every other Phase 3 plan attaches to this saga as a consumer, and every other plan's automated tests rely on the test projects Wave 0 creates. Without Wave 0 test scaffolding + the state machine landing first, nothing else can be wired. This is the Wave 1 blocker plan.

Output: Test project skeletons + shared fixtures, SagaEvents.cs, SagaCommands.cs, BookingSagaState.cs, BookingSaga.cs, BookingSagaDefinition.cs, SagaDeadLetter.cs, BookingDbContext (extended), migration (incl. Warn24HSent + Warn2HSent columns), BookingsController.cs, Program.cs (extended), and unit tests using MassTransit.TestFramework.
</objective>
